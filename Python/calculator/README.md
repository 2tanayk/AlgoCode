## Calculator

This is a simple calculator with this functionalities (`+`, `-`, `/`, `*`),

### Usage:
```sh
~ $ python3 main.py 3 + 3
6
```

```sh
~ $ python3 main.py 3 - 3
0
```

```sh
~ $ python3 main.py 3 / 3
1
```

```sh
~ $ python3 main.py 3 * 3
9
```

__Note__: will implement other operational methods to make more strong this calculator.
