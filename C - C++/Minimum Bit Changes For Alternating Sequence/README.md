# Problem Statement
Find the minimum number of bits to toggle (change 1 to 0 and vice versa) so that the resulting sequence of string would be alternating in nature (either 01010... or 10101...).

### Samaple Input 1
011101
### Sample Output 1
1

### Sample Input 2 
100101000
### Sample Output 2 
3
